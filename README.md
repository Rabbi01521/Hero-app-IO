## 📖 Description

A modern and responsive app marketplace where users can explore, search, install, and manage applications easily. The platform is designed with a clean UI, smooth user experience, and powerful features like live search, sorting, and data persistence.

Users can view detailed app information, analyze ratings through charts, and manage installed apps with real-time updates using localStorage.

---

## ✨ Key Features

- 🔍 Live Search (case-insensitive)
- 📱 Fully Responsive Design
- 📊 App Details with Interactive Charts
- ⭐ Ratings & Reviews Visualization
- ⬇️ Install & Uninstall System
- 💾 LocalStorage Data Persistence
- 🔽 Sort by Downloads (High → Low / Low → High)
- ⚡ Loading Animation & Smooth UI
- 🚫 Custom Error Page
